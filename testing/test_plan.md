# Планирование тестирования

## Функциональное тестирование
 Проверить корректность отображения карточек товаров
 Убедиться в правильности работы корзины
 Проверить добавление/удаление товаров из корзины
 Тестирование расчета общей суммы заказа

## Пользовательское тестирование 
 Оценить удобство навигации по каталогу
 Проверить интуитивность интерфейса корзины
 Выявить сложности в процессе покупки
 Собрать предложения по улучшению

## Тестирование производительности
 Проверить время загрузки каталога товаров
 Тестирование отзывчивости интерфейса корзины
 Проверить работу при большом количестве товаров

## Область тестирования

### Включено в тестирование:
 Компоненты карточек товаров
 Компоненты корзины покупок
 Система управления состоянием корзины
 Навигация между страницами каталога
