# React clothing store

# React clothing store - это веб приложение магазина одежды состоящий из множества карточек в онлайн магазине. Оно позволяет добавлять и удалять из карзины товар который пользователь добовляет с помощью кнопки "Добавить".
# Проект создан для дальнейшей разработки  Веб-приложения онлайн магазина одежды на подобие Lamoda, Ozon и глория джинс

Ключевые Функции 
* Добавление карточки в корзину
* Удаление Карточек из корзины
* Увелечение количества товара в корзине с помощью кнопок интерфейса 

# Установка и запуск
Следуйте этим шагам, чтобы запустить проект на вашем локальном компьютере.

# Предварительные требования
Убедитесь, что на вашем компьютере установлены:
* Node.js(версия 16 или выше)
* npm (обычно устанавливается вместе с Node.js)

# Инструкция по установке
1. Клонируйте репозиторий
    bash
    git clone https://github.com/Gojl11/kt-.-
2. Перейдите в директорию проекта
    bash
    cd react-task-tracker
3. Установите зависимости
    bash
    npm install
4. Запустите приложение в режиме разработки
    bash
    npm start
5. Откройте браузер
    Приложение автоматически откроется по адресу [http://localhost:3000](http://localhost:3000).

# Примеры использования

# Добавление новой карточки в корзину
1.  Выберете карточку твоара 
2.  Нажмите кнопку добавить кновку добавить в корзину
3.  Новый товар появляется в корзине слева
4.  Манипулируетйте количеством товара с помощью кнопок интерфейса
